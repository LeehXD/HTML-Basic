<!DOCTYPE html >

Está associado ao DTD = Document Type Declaration, é uma maneira de informar ao navegador o tipo de documento e garantir que esse documento seja manipulado da mesma maneira em diferentes navegadores.>

<html lang="pt-br">

Usada para que os user-agents saibam qual a linguagem principal do documento

<head>

É a parte que não é exibida no navegador da Web quando a página é carregada. Ele contém informações como Titulo e Icone.

<body>

Contem todo conteudo do web site.

<h1>

Niveis de titulos de h1 até h6.

<p>

Paragrafo.

<code>

Usada para apresentar uma linguagem em formato de código.

<strong>

Colocar a letra em negrito.

<em>

Colocar a letra em itálico.

<pre>

Organiza da mesma forma que é apresentado no código, formata o texto.
